Networking & The Internet Terminology

Term Definition
Networking Classification
    LAN
    MAN
    WAN
    Local Area Network (LAN) - Network that connects computer and devices in a limited geographical area such as home, school library or an office building.
    Metropolitan Area Network (MAN) - A MAN is a computer that is larger than a LAN. The area covered could range a few city blocks to the entire city.
    Wide Area Network (WAN) - A WAN is a network that covers a very large area such as one that goes across metropolitan, national, and international boundaries.
Network Model
    bus network
    star network
    A bus network is a server, clients, main cable, terminators at each end
    A star network is a at home wifi server that is a hub/switch, clients, server
Protocols
    CSMA/CD Ethernet
    CSMA/CA Wifi
    CSMA/CD means "Carrier sense multiple access with collision detection" and it is an informal conversation in which each person speaks when the opportunity presents itself, but only one person speaks at any time
    CSMA/CA means "Carrier Sense Multiple Access/ Collision Avoidance" which is for wireless networks

Internet Addressing
    Tier 1
    Tier 2
    Tier 3
 The Internet is broken down into groups called tiers.
    Tier 1 consists of a small number of companies called Tier 1 providers. The Tier 1 providers own long-distance, high-speed fiber-optic networks called backbones. These backbones span the major cities of the earth (not all Tier backbones go to all cities) and interconnect at special locations called network access points (NAPs). Anyone wishing to connect to any of the Tier 1 providers must pay money. The Tier 1 providers do not charge each other to connect.
    Tier 2 providers own smaller, regional networks and must pay the Tier 1 providers. Most of the famous companies that provide Internet access to the general public are Tier 2 providers.
    Tier 3 providers are even more regional and connect to Tier 2 providers.

Internet Protocols
    FTP
    HTTP
    SMTP
    The ports for FTP are TCP/20 and TCP/21. FTP stands for File Transfer Protocol, and its function is sends and receives files between systems
    The port for HTTP is TCP/80. HTTP stands for Hypertext Transfer Protocol, and its function is web server communication
    The port for SMTP is TCP/25. SMTP stands for Simple Mail Transfer Protocol, and its function is transfer e-mail between servers (sending)
World Wide Web
    URL
    Client-side Activities
    Server-side Activities
    URL stands for Uniform Resource Locator and it is a unique address for each document; contains the information needed by a browser to contact the proper server and request the desired document
    Client-side activities are activities that are performed by the client (such as a browser)
    Server-side activities are activities that are performed by the server (such as a Web server)
Internet four-layer software
    Application Layer
    Transport Layer
    Network Layer
    Link Layer
    The application layer is responsible for providing services to the user
    The transport layer is responsible for delivery of a message from one process to another
    The network layer is responsible for the delivery of packets from the original source to the final destination
    The link layer is responsible for transmitting frames from one node to the next
Internet is:
    Packet-switching Network
    Fault Tolerant
    Redundant
    Packet switching is the approach used by some computer network protocols to deliver data across a local or long distance connection
    Fault tolerance is the property that enables a system to continue operating properly in the event of the failure of (or one or more faults within) some of its components
    When a routing system is "redundant" it is when data is sent from one part of the network to another, the data is broken into block of bits (packets) that contain enough information so that each packet can be rerouted if necessary. redundancy is a good thing, in case some routers fail because then packets will be rerouted and still successfully send.
Cybersecurity
    Malware
    Virus
    Worm
    Trojan Horse
    Spyware
    Phishing
    DDoS attack
    Malware is software that is installed on a computer system with the intention to cause damage or steal information
    A virus is a malware that propagates itself by inserting a copy of itself into another program. Almost all viruses are attached to an executable file, which means they will not be active or be able to spread until a user runs or opens the host file or program. Viruses spread when the software or document they are attached to is transferred from one computer to another.
    Worms are similar to viruses in that they replicate functional copies of themselves, but are standalone software that do not require a host program to propagate. To spread, worms exploit a vulnerability on the target system or use social engineering to trick users into executing them.
    A trojan horse is disguised as legitimate software, a harmful piece of software that tricks users into loading and executing it on their systems.
    Spyware is software that enables a user to obtain covert information about another's computer activities by secretly transmitting data from their hard drive.
    Phishing is when attackers use fake emails or websites, appearing to come from a known and trustworthy source, in an attempt to gather personal or financial information from the recipients.
    A (distributed) denial of service attack that targets websites and web servers with the intention of bringing the services they provide to a halt. If one host is involved in the attack, the attack is considered to be a DoS attack. If multiple hosts are involved, it is a DDoS attack.
Protection
    Firewall
    Spam Filter
    Proxy Server
    Antivirus Software
    Cryptography
    Public Key Encryption
    A firewall is a network security device (hardware, software, or both) that grants or rejects network access to traffic flows between an untrusted zone and a trusted zone based on a set of rules.
    A spam filter is a program used to detect unsolicited, unwanted and virus-infected emails and prevent those messages from getting to a user's inbox
    A proxy server is an intermediary server that retrieves data from an Internet source, such as a webpage, on behalf of a user.
    Antivirus software is an application that is designed to check computers for viruses and remove them if found. They often also scan all incoming and outgoing data for suspicious code.

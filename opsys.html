Operating Systems Terminology

Term Definition
Operating System The software that controls the overall operation of the computer including file managment, the user interface, and the program execution environment.
Job Queue A storage organization in which objrcts are ordred in first-ins, first-outs
FIFO First In First Out
Interactive Processing a feature that allowed a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing a feature that provided service to multiple users at the same time
Multiprogramming A technique applied to implement time-sharing. time is divided into intervals and then the execution of each job is restricted to only one interval at a time. By rapidly shuffling the jobs back and forth in this manner, the illusion of several jobs executing simultaneously is created.
Application Software consists of the programs for performing tasks particular to the machine's utilization. Software that performs some useful task such as word processing or playing a game for a user.
System Software In contrast to application software, perform those tasks that are common to computer systems in general. In a sense, the system software provides the infrastructure that the application software requires, in much the same manner as a nations infrastructure(Governmentm roads, utilities, financial instructions). Within the class of system software there are two categories; operating system and utility software
Utility Software consists of programs for performing activities that are fundamental to computer installations but not included in the operating system. Consists of software units that extend the capabilities of the operating system.Examples copy a file from a magnetic disc to flash drive, compress and decompress data, playing multimedia content, handling network communication.
Graphical User Interface (GUI) More modern systems use this. objects to be manipulated, such as files and programs, are represented pictorially on the display as icons. For example a computer mouse, with one or more buttons can be used to click or drag icons on the screen.
Shell older user interfaces communicated with users through textual messages using a keyboard and monitor screen
Kernel the internal part of an operating system that contains those software components that perform basic functions required by the computer installation. Such units are the file manager, the device driver, the memory manager, the scheduler and the dispatcher.
Device Driver operates or controls a particular type of device that is attached to a computer. Each device driver is uniquely designed for its particular type of device(such as a printer, disc drive or monitor). The result is a generic(overall) operating system that can be customized for particular peripheral(yttre) devices by merely installing the appropriate device drivers.
File Manager coordinates the use of the machine's mass storage facilities. Maintains records of all the files stored in mass storage, including where each file is located, which users are allowed to access the various files.
Memory Manager coordinating the machine's use of main memory
Virtual Manager a tool or resource that allows for the management of more than one computer from a single workstation
Boot Strapping How the operating system gets started. Performed by a computer every time its turned on. Transfers the operating system from mass storage(where its permanently stored) into main memory(which is essentially empty when the machine is first turned on.
ROM a small portion of a computer's main memory where the CPU expects to find its initial program is constructed from special nonvolatile memory cells. containing non-volatile, permanent data that, normally, can only be read, not written to. ROM contains the programming that allows a computer to start up or regenerate each time it is turned on. Main memory är ju tomt när datorn startas och datorn behöver instruktioner från main memory för att något ska hända, därför har man gjort på detta vis. The program that is initially executed when the machine is turned on is called the boot loader, the instructions in the boot loader direct the CPU to transfer the operating system from its location into volatile area of main memory.
Deadlock the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling olding data for output at a later more convenient time. A technique for granting several processes access to a common resource. It makes processes think they have access to the for ex. printer, when it actually stores the information in I queue in mass storage so the process can continue and not be waiting.
Super User (or Administrator) A user that is authorized (and therefore, trusted) to perform security-relevant functions that ordinary users are not authorized to perform.
Multi-Factor Authentication An authentication system that requires more than one distinct authentication factor for successful authentication. Multifactor authentication can be performed using a multifactor authenticator or by a combination of authenticators that provide different factors. The three authentication factors are something you know, something you have, and something you are.
